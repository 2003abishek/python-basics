# Data Structures Types.

## Tree Non Linear Type

| Tree Type      | Insert    | Delete    | Traversal |
|----------------|-----------|-----------|-----------|
| Binary Tree    | O(n)      | O(n)      | O(n)      |
| AVL Tree       | O(log n)  | O(log n)  | O(n)      |
| Red-Black Tree | O(log n)  | O(log n)  | O(n)      |
